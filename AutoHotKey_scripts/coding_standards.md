Coding standards and preferences

- Always use Allman-style curly braces for all blocks (even single-statement blocks).
- Use two-space indentation (spaces, not tabs).
- Align assignments vertically where practical.
- Put a single space on either side of parameters inside parentheses, e.g. Func( a, b ).
- Always use parentheses for control statements and function calls, examples:
  -  if( x > 0 )
  -  Func( a, b )
- Never put a space between a control statement keyword or function name and it's opening parenthesis, examples:
  - Func( a, b ), not Func ( a, b )
  - if( x > 0 ), not if ( x > 0 )
- Always wrap blocks in curly braces, even for single statements.
- Prefer the Prefix operator over the Postfix operator wherever possible.
